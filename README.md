# 📧 Spam Detection

An intelligent **Spam Detection Web Application** built with **Python** and **Flask** that uses **Machine Learning** and **Natural Language Processing (NLP)** to classify text messages or emails as **Spam** or **Not Spam (Ham)**. The application helps users quickly identify unwanted or malicious messages through an easy-to-use web interface.

> **Detect spam instantly using the power of Artificial Intelligence and Machine Learning.**

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/subham-paul/Spam_Detection.git
```

```bash
cd Spam_Detection
```

---

## 2. Create a Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate:

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
```

Activate:

```bash
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Run the Application

```bash
python app.py
```

or

```bash
flask run
```

# 📝 How It Works

1. Enter an email or SMS message.
2. The text is cleaned and preprocessed.
3. NLP techniques extract important features.
4. The trained Machine Learning model analyzes the text.
5. The application predicts whether the message is **Spam** or **Not Spam (Ham)**.
6. The prediction is displayed instantly on the web interface.
